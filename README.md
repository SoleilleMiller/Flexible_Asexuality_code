# Flexible_Asexuality_code
The data and R code for the analysis of genetic data as recorded in "Flexible Asexuality: Naturally occurring variation in mechanisms of parthenogenesis within lineages and individuals of a facultative parthenogen, Megacrania batesii"



List of files: 
./organizing_data/metadata_in_progress.v.1.csv - Metadata sheet
Report_DMegac23-8515_moreOrders_SNP_1.csv - Genetic data (SNPs) recieved from DArT.
./Comparing_Sample_Duplicates/removal_list.RDS - List of sample duplicates that were removed from analysis. 
data_for_linegraph_all.v.1.csv - Metadata for linegraphs 
sa_clean_FIXed.csv - Metadata for S-A reproductive regime 
saa_clean_full.csv - Metadata for S-A-A reproductive regime 
aaa_loh_add_matriline_DONE.csv - Metadata for A-A-A reproductive regime 
sss_loh_add_matriline_DONE.csv - Metadata for S-S-S reproductive regime
